# Boot Linux Kernel

## Introduction

1. Power button pressed.

2. `BIOS`/`UEFI` engaged.

3. `Bootloader` loaded and given control.

4. `RAM disk filesystem created`.

5. `Kernel` unpacked, and, hardware driver modules loaded.

6. `Kernel` started and takes control.
